72 Melakarta Ragas

The 72 Melakarta Ragas form the backbone of the Carnatic music system, serving as parent ragas from which other ragas (Janya ragas) are derived. Each Melakarta raga consists of seven notes (swaras) in both ascending (Arohana) and descending (Avarohana) scales, and they are organized into 12 chakras (groups), with six ragas in each chakra.

1. Indu Chakra

  1. Kanakangi
  2. Ratnangi
  3. Ganamurti
  4. Vanaspati
  5. Manavati
  6. Tanarupi

2. Netra Chakra

  1. Senavati
  2. Hanumatodi (commonly known as Todi)
  3. Dhenuka
  4. Natakapriya
  5. Kokilapriya
  6. Rupavati

3. Agni Chakra

  1. Gayakapriya
  2. Vakulabharanam
  3. Mayamalavagowla
  4. Chakravakam
  5. Suryakantham
  6. Hatakambari

4. Veda Chakra

  1. Jhankaradhvani
  2. Natabhairavi
  3. Keeravani
  4. Kharaharapriya
  5. Gourimanohari
  6. Varunapriya

5. Bana Chakra

  1. Mararanjani
  2. Charukesi
  3. Sarasangi
  4. Harikambhoji
  5. Dheerasankarabharanam
  6. Naganandini

6. Ritu Chakra

  1. Yagapriya
  2. Ragavardhini
  3. Gangeyabhushani
  4. Vagadheeswari
  5. Sulini
  6. Chalanata

7. Rishi Chakra

  1. Salagam
  2. Jalarnavam
  3. Jhalavarali
  4. Navaneetam
  5. Raghupriya
  6. Pavani

8. Vasu Chakra

  1. Gavambhodi
  2. Bhavapriya
  3. Shubhapantuvarali
  4. Suvarnangi
  5. Divyamani
  6. Shadvidhamargini

9. Brahma Chakra

  1. Dhavalambari
  2. Namanarayani
  3. Kamavardhini
  4. Ramapriya
  5. Gamanashrama
  6. Vishwambari

10. Disi Chakra

  1. Syamalangi
  2. Shanmukhapriya
  3. Simhendramadhyamam
  4. Hemavati
  5. Dharmavati
  6. Neetimati

11. Rudra Chakra

  1. Kantamani
  2. Rishabhapriya
  3. Latangi
  4. Vachaspati
  5. Mechakalyani
  6. Chitrambari

12. Aditya Chakra

  1. Sucharitra
  2. Jyotiswarupini
  3. Dhatuvardhani
  4. Nasikabhushani
  5. Kosalam
  6. Rasikapriya
